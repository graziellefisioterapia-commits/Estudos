CSS externo

Um CSS externo, como o próprio nome sugere, está fora do documento HTML.

As principais vantagens no uso desse método são:

  • Como o arquivo está "externo", ou seja fora do arquivo HTML o seu conteúdo fica separado do estilo, e isso deixa tudo mais organizado.
  • Ainda podemos permanecer aplicando os diversos estilos em cascata. Ainda sendo possível aplicar as duas outrasestruturas.
  • Um mesmo arquivo de estilo pode ser utilizado em várias páginas do seu sistema site. Vamos aplicar também os estilo na home do nosso exercício.
